## Зачем?
Данный генератор предназначен для простого описания моделей на целевом языке. На данный момент поддерживается только язык **Golang**.

Если вы хотите поучаствовать в тестировании, заполните [форму](https://forms.gle/XibxArXcEEV5NRwa6).
